# Eaglercraft 1.8.8 웹버전

이 프로젝트는 웹 브라우저에서 실제 마인크래프트 자바 1.8.8 버전을 플레이할 수 있게 만들어진 정적 웹 기반 Eaglercraft 게임입니다.

## 개발 시 참고사항

- **로컬 서버 실행 필수**: 게임 코드 내에서 `file://` 프로토콜을 통한 직접 실행을 차단하고 있습니다. 따라서 브라우저에서 로컬 파일을 바로 열면 안 되며, 반드시 로컬 HTTP 서버를 실행하여 접속해야 합니다.
  - 실행 예시: 터미널에서 `python3 -m http.server` 명령어를 입력한 뒤, 브라우저에서 `http://localhost:8000` 등의 주소로 접속하여 플레이하세요.
