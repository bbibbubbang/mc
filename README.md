# Eaglercraft 1.8.8 웹버전

이 프로젝트는 웹 브라우저에서 실제 마인크래프트 자바 1.8.8 버전을 플레이할 수 있게 만들어진 정적 웹 기반 Eaglercraft 게임입니다.

## 플레이 방법

- **웹 호스팅 사용 (권장)**: GitHub Pages (`.github.io`)와 같은 정적 웹 호스팅 서비스에 이 프로젝트 파일들을 업로드하면 브라우저를 통해 바로 접속하여 플레이할 수 있습니다.

## 개발 및 로컬 테스트 시 참고사항

- **로컬 서버 실행 필수**: 게임 코드 내에서 `file://` 프로토콜을 통한 직접 실행을 차단하고 있습니다. 따라서 내 컴퓨터에서 브라우저로 로컬 파일을 더블 클릭해 바로 열면 안 되며, 반드시 로컬 HTTP 서버를 실행하여 접속해야 합니다.
  - 실행 예시: 터미널에서 `python3 -m http.server` 명령어를 입력한 뒤, 브라우저에서 `http://localhost:8000` 등의 주소로 접속하여 테스트하세요.

## 모바일 환경 플레이 지원

모바일 기기에서 접속 시, 게임 화면 상단에 투명한 터치 컨트롤러 오버레이가 나타나 게임을 조작할 수 있습니다.
- 화면 회전(Landscape)이 지원됩니다.
- 좌측 하단에 고정된 360도 이동 패드를 제공합니다.
- 우측 하단에 JUMP(점프), SHIFT(웅크리기 토글), L-CLK(좌클릭), R-CLK(우클릭) 버튼이 제공됩니다.
